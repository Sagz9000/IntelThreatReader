# Intel Threat Reader

A cybersecurity threat intelligence dashboard that ingests news feeds (RSS), analyzes them using local AI (Ollama), and presents categorized risk assessments.

## UI Components & Features

### 1. Intelligence Dashboard
The main landing page categorizes threat reports by type (Ransomware, Malware, etc.) and risk level. 
- **Dynamic Grid**: Responsive card layout for quick scanning of threat landscapes.
- **Source Attribution**: Every story is tagged with its origin (e.g., BleepingComputer, Krebs on Security).
- **Custom User Tags**: Manually triage stories by adding your own tags directly onto cards.

![Dashboard Main](C:\Users\potte\.gemini\antigravity\brain\983a56c7-b3d6-47ab-8eab-08544ac2dd27\dashboard_main_1767750070806.png)

### 2. Article Deep-Dive
Go beyond the summary with full content extraction and AI-powered insights.
- **Executive Summary**: Concise 2-sentence summary generated by Llama 3 for rapid briefing.
- **Multimedia Evidence**: Scraped images, charts, and diagrams from the original source for full context.
- **Risk Assessment**: Automatic classification based on severity.

![Article Details](C:\Users\potte\.gemini\antigravity\brain\983a56c7-b3d6-47ab-8eab-08544ac2dd27\article_details_view_1767750096059.png)

### 3. AI Analyst Chat (RAG)
A persistent, context-aware chatbot available across all pages.
- **Vision Capable**: Understands charts and images retrieved from stories.
- **IOC Defanging**: Securely presents indicators (IPs, URLs) by defanging them (e.g., `hxxp`, `[.]`).
- **Indicator Export**: One-click extraction and clipboard export for technical indicators.
- **Minimize to Bubble**: Stay focused with a persistent floating bubble mode.

### 4. Feed Management
Customize your intelligence feed sources with ease.
- **Add/Remove Feeds**: Paste any RSS/Atom link to start monitoring.
- **Dynamic Toggle**: Instantly enable or disable sources as needed.

## Setup & Run

### Prerequisites
- Docker & Docker Compose
- NVIDIA GPU (Recommended for Ollama)

### Installation
1. Clone the repo.
2. Run `docker-compose up -d --build`.
3. Access the dashboard at `http://localhost:5001`.

### Usage
- Use the **Refresh Feeds** button to fetch and analyze new articles. *Note: The first run will take some time as it pulls 3GB+ of models.*
- Click on an article to see the AI summary and full details.
- Use the **Chat Widget** in the bottom right to ask questions like "What are the latest ransomware attacks?".

## Tech Stack
- **Backend**: Flask, BeautifulSoup4 (Scraper), APScheduler (Background Analysis)
- **AI Engine**: Ollama (Llama 3.2 Vision), ChromaDB (Multimodal Vector Store)
- **Frontend**: Vanilla CSS3 (Glassmorphism), JavaScript (Persistent State)
- **Database**: SQLite (Metadata & User Tags)
